### 🗺️ [Siwa — Booking Engine](https://github.com/Sayed-Herzallah/Siwa-Oasis-APi)

**`PROBLEM`** Double-booking race conditions during peak traffic causing financial errors.

**`SOLUTION`**
- InnoDB row-level locking with `SELECT ... FOR UPDATE`
- 100% elimination of double-booking under concurrent load
- Multi-entity relational schema with cross-entity checks
- Cursor-based pagination on large datasets
